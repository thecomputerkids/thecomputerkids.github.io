Why Teach Code?

  1. Code is one of the world's most widely used languages
    2. Computer science is the new language of the world, and it’s also one of the fastest growing occupations. While it would be foolish to assume that we know exactly what the jobs of the future will be, we do know that 42 per cent of the Canadian labour force is at high risk of being affected by automation.
  2. Connected devices are already reshaping our world
    3. Kids also need to learn how to code because computers and connected devices are what will shape their world as they grow up, just as physics, Math, Chemistry and Biology shape our world right now. In about twenty years, an inability to code or at least to understand logic-based thinking to some level will be just as crippling as illiteracy and innumeracy is today.
  3. Kids learn better and faster when they're young
    4. Teaching your child how to code when they are still young is important, because kids have an easier time learning skills than adults do – their minds are flexible and open, and learning code is like learning a language, far easier when you are young.
  4. Coding is creativity unleashed
    5. Just like art and craft is a way to express creativity, coding can be a highly engaging, fun and empowering skill for kids today. Platforms like Scratch , teach kids how to code in a way that doesn’t make them feel like they are stuck in a classroom learning something they don’t want to. They start out by playing games, and if the child doesn’t like a particular aspect of the game, they can modify it to their preference by rewriting the code.
  5. Problem solving and critical thought
    6. Coding techniques also helps children to develop critical thinking skills and problem solving processes that are not only important in computer science, but also in life. It teaches them how to look at the bigger picture, and to break down big challenges into smaller and more manageable tasks.

Programs

I want to teach code in a fun and friendly way! All programs will be tailored toward your childs interests and learning abilites.

HTML & CSS

This beginner friendly program will focus on reading and writing HTML & CSS. Using a tool called Atom and Mozilla Thimble children will develop an understanding of HTML and CSS by adding links, images, videos and will create their very own web page from scratch! During this course, students will also complete various code challenges on a white board to ensure the knowledge is sticking.

Animating with Scratch

Using Scratch a beginner level programming environment, students will create interactive stories and games through animation. This program is targeted toward younger children looking to get comfortable navigating the keyboard and internet.

Java Basics

Java is a widely used coding language and is used in about everything, from web applications to toasters. During this program students will have an understanding of data types & variables, operators and statements. This course is targeted toward an older student with a strong interest in learning code.
